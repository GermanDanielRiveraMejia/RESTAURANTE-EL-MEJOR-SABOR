**# Menú de Restaurante – Kivy**



\## ¿Qué hace la app?



La aplicación muestra el menú de un restaurante de forma visual.  

Permite ver las categorías de productos y, al seleccionar una categoría, muestra los productos correspondientes con su nombre y precio.

Los datos del menú se cargan desde un archivo JSON externo.
